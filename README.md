# Online_Banking_Project_java
✅ Features
🆕 Open Account — Savings or Current, with 4-digit PIN
🔐 Secure Login — PIN-based authentication
💰 Deposit & Withdraw — With balance validation
🔁 Fund Transfer — Between any two accounts
📋 Transaction Statement — Full history with date & time
🔒 Account Lock — Auto-locks after 3 wrong PIN attempts
🛠️ Admin Panel — View all accounts (password protected)
